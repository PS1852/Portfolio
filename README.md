# Pranjal's Luxury Portfolio

A high-end, cinematic portfolio experience built with React 19, GSAP, and Three.js.

## Features
- Cinematic Loading Sequence
- Custom Physics Cursor
- Horizontal Project Track
- Editorial Typography
- Fully Responsive Design

## Tech Stack
- **Frontend:** React 19, Vite, Tailwind CSS
- **Animations:** GSAP 3, Framer Motion, Lenis
- **Deployment:** GitHub Actions

## Development
```bash
npm install
npm run dev
```

## Production
```bash
npm run build
```
